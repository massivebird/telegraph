A CLI for practicing morse code!

Written in Rust 🦀

Currently only supports straight-key style (`Spacebar`).

⚠️ Requires your terminal emulator to emit keypress-release events. Tested on [kitty](https://github.com/kovidgoyal/kitty) version `>= 0.4.5`.
